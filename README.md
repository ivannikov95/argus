# Argus

Локальный инструмент для воспроизводимой биомедицинской статистики. Загружаешь CSV/XLSX с клиническими данными — получаешь автоматическую таблицу с правильными критериями и экспорт в DOCX готовый к публикации.

## Стек

- **Backend:** Python / FastAPI / pandas / scipy / python-docx
- **Frontend:** React 18 / TypeScript / Vite
- **Хранение:** локальные JSON-файлы в `data/projects/`

## Запуск

```bash
# Backend (порт 8000)
./venv/bin/python run_server.py

# Frontend (порт 5173, в отдельном терминале)
cd frontend && npm run dev
```

Открой `http://127.0.0.1:5173`.

## Что умеет прямо сейчас

| Функция | Статус |
|---|---|
