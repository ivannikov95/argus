# Roadmap

## Сделано

- Главная страница — дашборд с последними проектами, секция возможностей
- Многослайдовая архитектура Table 1: каждая таблица — отдельная «страница» со своими настройками, переключение стрелками
- Автосохранение (Google Docs–style): изменения пишутся в тот же файл через 3 сек; «Сохранить как» — явно создаёт новый проект
- Инлайн-редактирование названия таблицы (в заголовке страницы и в бумаге)
- Переключатель столбца «Все» (showOverall)
- Упрощённые подписи строк: «Возраст, Me [Q1;Q3]» вместо полного названия теста
- Фикс переноса текста в ячейках таблицы
- DOCX с несколькими таблицами и разрывами страниц
- Кастомные подписи переменных прокидываются в DOCX

## Известные баги / долги

- `stats.f_oneway` не принимает `equal_var` → Welch ANOVA для >2 групп упадёт с TypeError (сейчас интерфейс ограничен двумя группами, но если добавить — надо реализовать через `pingouin` или вручную)
- `confidence_level` захардкожен 0.95 в `api.ts` — нет UI-контрола
- При загрузке проекта восстанавливается только первый слайд (last_analysis); остальные слайды не сохраняются в JSON

## Модули в очереди (отключены в nav)

- **Сравнение групп** — множественные сравнения, post-hoc тесты (Tukey, Dunn)
- **Корреляции** — матрица корреляций (Pearson / Spearman), тепловая карта
- **Регрессия** — логистическая и линейная, таблица коэффициентов с OR/β и ДИ
- **Выживаемость** — Kaplan–Meier, лог-ранк тест, кривые в DOCX
