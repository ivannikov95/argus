# Roadmap

## Сделано

- Главная страница — дашборд с последними проектами, секция возможностей
- Многослайдовая архитектура Table 1: каждая таблица — отдельная «страница» со своими настройками, переключение стрелками
- Автосохранение (Google Docs–style): изменения пишутся в тот же файл через 3 сек; «Сохранить как» — явно создаёт новый проект
- Инлайн-редактирование названия таблицы (в заголовке страницы и в бумаге)
- Переключатель столбца «Все» (showOverall)
- Упрощённые подписи строк: «Возраст, Me [Q1;Q3]» вместо полного названия теста
- DOCX с несколькими таблицами и разрывами страниц; кастомные подписи переменных
- Датасет: пагинация (50 строк/стр), инлайн-редактирование ячеек, нумерация строк
- Тост-уведомления: исчезают через 5 сек с анимацией
- Удаление проекта из дашборда; идемпотентное DELETE на бэкенде
- Защита от дубликатов: «Сохранить» скрыт когда проект уже отслеживается
- Полуавтоматический расчёт Table 1: кнопка убрана, пересчёт через 0.7 сек после выбора переменной
- Пустой стартовый лист Table 1: переменные не выбраны по умолчанию
- Client-side routing (`react-router-dom`): каждая страница имеет свой URL (`/project/:id/table` и т.д.), рефреш сохраняет позицию
- Кнопка-логотип M в топбаре → переход на главную; убран лишний дропдаун «Открыть проект»
- Страница переменных: убраны галочки «В анализ» (дублировали выбор в инспекторе Table 1)
- **Типы переменных теперь работают**: изменение числовой → категориальную в словаре реально меняет метод анализа — `variable_overrides` передаются в API и применяются на бэкенде

## Известные баги / долги

- `stats.f_oneway` не принимает `equal_var` → Welch ANOVA для >2 групп упадёт с TypeError (сейчас интерфейс ограничен двумя группами)
- При навигации браузером «назад/вперёд» URL меняется, но страница не реагирует (нет `popstate` listener)

## Модули в очереди (отключены в nav)

- **Сравнение групп** — множественные сравнения, post-hoc тесты (Tukey, Dunn)
- **Корреляции** — матрица корреляций (Pearson / Spearman), тепловая карта
- **Регрессия** — логистическая и линейная, таблица коэффициентов с OR/β и ДИ
- **Выживаемость** — Kaplan–Meier, лог-ранк тест, кривые в DOCX
